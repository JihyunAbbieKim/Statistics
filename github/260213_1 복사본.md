***Using Big Data to Emulate a Target Trial When a Randomized Trial Is Not Available***

**[Protocol of a Target Trial Emulation]**

1. **Eligibility criteria**

   데이터의 제약, follow-up 실패로 인한 문제로 모방에 실패할 수 O

   \- Inclusion에 제약을 두면 Selection Bias
   $$
   \small\text{\color{gray}병원 데이터(EMR)가 완벽한 환자만 포함(S)한다면}\\
   \small\text{\color{gray} 증상이 심한 사람(Y)은 검사를 많이 해서 데이터가 완벽할 가능성이 높고, 특정 약물(A)을 처방받은 사람도 모니터링이 잦아 데이터가 완벽할 수 있음.}\\
   \small\text{\color{gray}결과적으로 원래 약물과 질병 사이에 아무런 관계가 없더라도, 데이터가 완벽한 사람들(S=1)만 모아놓고 보면 약물과 질병이 연관된 것처럼 보이는 편향이 생김.}
   $$
   

   \- follow-up이 중단된 데이터를 삭제하면 Selection Bias ∴ 중도 절단(censoring)으로 처리

   

2. **Teatment strategies**

   지금까지의 논문들이 initiator(new-user)를 고수한 이유

   \- 기존 사용자(users) 중 취약한 사람들이 부작용 등으로 복용을 중단하여 기존 users 집단은 강한 사람들만 선택되어 구성되는 자기 선택 편향이 발생할 수 있음

   i) Propensity score란?
   $$
   \text{Propensity score는 IP weighting, g-estimation할 때 사용되는 } \pi(L)\\
   \text{측정 가능한 공변량 L에 대해 추정된 성향점수를 } \pi(L) \text{이라고 하자.}\\
   \text{가장 이상적인 randomized trial은 } \pi(L) = 0.5\text{ for all individuals}\\
   이때\, \pi(L)\text{의 참값을 모르므로 로지스틱 회귀 등으로 추정해야 함}\\
   \text{b(L) is any function of the covariates L s.t. }A\perp\!\perp\!L|b(L)\\
   \text{where A=1 or A=0}\\
   \pi(L) \in b(L)
   $$

**타겟 시도 모사(TTE)에서의 구현**

관찰 데이터를 활용한 타겟 시도 모사에서는 RCT와 같은 '강제적 배정'이 없기 때문에, 위 두 가지 분석을 다음과 같이 모방함

**1. ITT 효과 모방: Initiator Design**

\- 모방 방법: 베이스라인(T0) 시점에 특정 치료를 처음 시작(Initiate)한 기록을 '무작위 배정'으로 간주

\- 특징: 일단 치료를 시작했다면, 그 이후에 약을 중단하거나 용량을 바꿔도 처음에 속했던 군의 데이터를 끝까지 추적

\- 한계: 처방은 받았으나 조제 자체를 하지 않은 'non-initiator'를 데이터에서 식별하기 어렵기 때문에(처방 =/=> 조제), RCT의 ITT보다는 다소 완화된 형태의 분석

**2. Per-Protocol 효과 모방: Cloning & Censoring**

\- 모방 방법: 대상자가 치료 지침에서 벗어나는 순간(약 중단, 타 약제 복용 등) 그 즉시 검열(Censoring) 처리하여 분석에서 제외

\- 복제(Cloning): 만약 '유예 기간(Grace Period)'이 있다면, 시작 시점에 대상자를 치료군과 대조군 양쪽에 모두 복제하여 배정했다가 지침에 어긋나는 시점에 한쪽 클론을 탈락시킴

\- 편향 보정: 지침 준수 여부가 환자의 상태에 영향을 받을 수 있으므로, 역확률 가중치(IPTW) 등을 사용하여 탈락한 사람들로 인해 깨진 군 간의 균형을 통계적으로 보정

#### 변수 선택 시 주의사항

| **보정하면 안 되는 변수** | **구조**  | **발생하는 문제**                             |
| ------------------------- | --------- | --------------------------------------------- |
| 매개 변수 (Mediator)      | A → M → Y | 전체 인과 효과를 왜곡하거나 과소평가함        |
| 콜라이더 (Collider)       | A → C ← Y | 없던 상관관계를 만들어내는 선택 편향 발생     |
| 결과 이후 변수            | Y → S     | 결과값 자체를 통제하여 추론을 불가능하게 만듦 |
